# FastGIF v1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship FastGIF v1.0 — export truth + zero-flicker Best mode + Origami motion + unified editor canvas — in 8 atomic commits, each independently shippable and each gated by `scripts/verify.sh`.

**Architecture:** Swift UI layer stays iA-Writer-minimal (one canvas, one overflow). Rust core grows a global-palette + temporal-Sierra2_4a path (`fastgif_encode_global`) plus a preview FFI (`fastgif_preview_frame`). Verification is proof-obligation-driven: four propositions from §9 of the spec become executable witnesses wired into `verify.sh`. Testing runs against a dedicated `FastGIF-Dev` simulator owned by this project — no sharing with other in-flight agents.

**Tech Stack:** Swift 5.10 / SwiftUI / Observation, Rust 2021 (`color_quant`, `gif`, `rayon`, `crossbeam-utils`), FlowDeck (xcodebuild wrapper), Maestro 2.1, `xcrun simctl`, `swift` script interpreter.

## Operating rules (read before any commit)

1. **Simulator isolation.** Every build/test in this plan targets the simulator named `FastGIF-Dev`, whose UDID is written to `scripts/.sim-udid` in commit 1. Never target a simulator whose name you did not create — other agents may be using shared sims. If `scripts/.sim-udid` is missing, re-run `scripts/sim-bootstrap.sh` before continuing.
2. **verify.sh is the gate.** After every commit below, run `scripts/verify.sh` and only commit when it is green. The script is the type-checker for §9's propositions.
3. **No premature testing.** Do **not** run `scripts/verify.sh` until it has been created in Task 1.6. Commits 1's earlier tasks build the infrastructure `verify.sh` needs; running it early is meaningless.
4. **LOC discipline check runs on commit 8 only.** `git diff --shortstat main...HEAD -- '*.swift'` must be non-positive net after commit 8. Earlier commits can be positive; the balance is a pass-level invariant, not a per-commit one.
5. **Commit message format:** `<scope>: <subject>` where `<scope>` is one of `infra`, `encoder`, `rust`, `editor`, `motion`. Plain, no emojis.
6. **Do not amend. Do not force-push. Do not `git add -A`.** Stage files by name.

```bash
#!/usr/bin/env bash
# Creates (or reuses) a dedicated FastGIF-Dev simulator.
# Writes the UDID to scripts/.sim-udid so every downstream script
# can target the same instance.
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
UDID_FILE="$SCRIPT_DIR/.sim-udid"
SIM_NAME="FastGIF-Dev"
DEVICE_TYPE="com.apple.CoreSimulator.SimDeviceType.iPhone-16-Pro"

# Find newest iOS runtime available
RUNTIME="$(xcrun simctl list runtimes -j \
    | /usr/bin/plutil -extract runtimes json -o - - \
    | /usr/bin/python3 -c '
import json, sys
data = json.load(sys.stdin)
ios = [r for r in data if r["platform"] == "iOS" and r["isAvailable"]]
ios.sort(key=lambda r: r["version"], reverse=True)
print(ios[0]["identifier"])' 2>/dev/null || true)"

if [[ -z "$RUNTIME" ]]; then
    echo "error: no iOS runtime available" >&2
    exit 1
fi

# Reuse existing FastGIF-Dev if present
EXISTING_UDID="$(xcrun simctl list devices "$SIM_NAME" -j \
    | /usr/bin/python3 -c '
import json, sys
data = json.load(sys.stdin)
for devices in data["devices"].values():
    for d in devices:
        if d["name"] == "FastGIF-Dev":
            print(d["udid"])
            sys.exit(0)' 2>/dev/null || true)"

if [[ -n "$EXISTING_UDID" ]]; then
    UDID="$EXISTING_UDID"
    echo "reusing existing FastGIF-Dev: $UDID"
else
    UDID="$(xcrun simctl create "$SIM_NAME" "$DEVICE_TYPE" "$RUNTIME")"
    echo "created FastGIF-Dev: $UDID"
fi

xcrun simctl boot "$UDID" 2>/dev/null || true
xcrun simctl bootstatus "$UDID" -b

echo "$UDID" > "$UDID_FILE"
echo "wrote $UDID_FILE"
```

This script writes a 3s/240×240/24fps synthetic gradient pan. We use a synthetic clip so the plan is reproducible across machines with no copyrighted source. It is deterministic.

```bash
#!/usr/bin/env bash
# Generates the canonical reference clip used by flicker baseline + perf bench.
# Output: tests/fixtures/cat-loaf-3s.mov  (3s, 240x240, 24fps, H.264)
set -euo pipefail

REPO="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
OUT="$REPO/tests/fixtures/cat-loaf-3s.mov"
mkdir -p "$(dirname "$OUT")"

/usr/bin/swift - <<'SWIFT'
import AVFoundation
import CoreImage

let url = URL(fileURLWithPath: ProcessInfo.processInfo.environment["OUT"]!)
try? FileManager.default.removeItem(at: url)

let W: Int = 240, H: Int = 240, FPS: Int32 = 24, DURATION: Int = 3
let writer = try AVAssetWriter(outputURL: url, fileType: .mov)
let settings: [String: Any] = [
    AVVideoCodecKey: AVVideoCodecType.h264,
    AVVideoWidthKey: W,
    AVVideoHeightKey: H
]
let input = AVAssetWriterInput(mediaType: .video, outputSettings: settings)
let adaptor = AVAssetWriterInputPixelBufferAdaptor(
    assetWriterInput: input,
    sourcePixelBufferAttributes: [
        kCVPixelBufferPixelFormatTypeKey as String: kCVPixelFormatType_32BGRA,
        kCVPixelBufferWidthKey as String: W,
        kCVPixelBufferHeightKey as String: H
    ]
)
writer.add(input)
writer.startWriting()
writer.startSession(atSourceTime: .zero)

let ctx = CIContext()
let frames = DURATION * Int(FPS)
for i in 0..<frames {
    while !input.isReadyForMoreMediaData { Thread.sleep(forTimeInterval: 0.005) }
    let t = Double(i) / Double(frames)
    // A slow horizontal color ramp that pans — guarantees gradient content,
    // which is where per-frame palette flicker is visible.
    let hue = t
    let ci = CIFilter(name: "CILinearGradient", parameters: [
        "inputPoint0": CIVector(x: 0, y: 0),
        "inputPoint1": CIVector(x: CGFloat(W), y: 0),
        "inputColor0": CIColor(red: hue, green: 0.2, blue: 1.0 - hue),
        "inputColor1": CIColor(red: 1.0 - hue, green: 0.8, blue: hue),
    ])!.outputImage!.cropped(to: CGRect(x: 0, y: 0, width: W, height: H))
    var pb: CVPixelBuffer?
    CVPixelBufferPoolCreatePixelBuffer(nil, adaptor.pixelBufferPool!, &pb)
    ctx.render(ci, to: pb!)
    let time = CMTime(value: CMTimeValue(i), timescale: FPS)
    adaptor.append(pb!, withPresentationTime: time)
}
input.markAsFinished()
let sem = DispatchSemaphore(value: 0)
writer.finishWriting { sem.signal() }
sem.wait()
if writer.status == .failed { FileHandle.standardError.write("writer failed: \(writer.error!)\n".data(using: .utf8)!); exit(1) }
print("wrote \(url.path)")
SWIFT
```

## Task 1.3 — `tests/validate_gif.swift` (binary validator + flicker metric)

**Files:**
- Create: `tests/validate_gif.swift`

- [ ] **Step 1: Write the validator**

```swift
#!/usr/bin/env swift
// Usage:
//   swift tests/validate_gif.swift <path-to-gif> [--expected-frames N]
//                                                [--expected-duration S]
//                                                [--mode best|good|draft]
//                                                [--baseline tests/fixtures/flicker-baseline.txt]
//                                                [--alpha 0.3]
//
// Exits non-zero with a diagnostic on any failed check.
//
// Flicker metric: reads palette indices in a 32x32 region centered at the
// midpoint, computes per-pixel variance across frames, averages them.
// This is `flicker(gif)` in §9.P2.

import Foundation
import ImageIO
import CoreImage
import CoreGraphics
import UniformTypeIdentifiers

func die(_ msg: String) -> Never {
    FileHandle.standardError.write((msg + "\n").data(using: .utf8)!)
    exit(1)
}

struct Args {
    var path: String
    var expectedFrames: Int = 24
    var expectedDuration: Double = 3.0
    var mode: String = "best"
    var baseline: String = "tests/fixtures/flicker-baseline.txt"
    var alpha: Double = 0.3
    var emitBaseline: Bool = false
}

var args = Args(path: "")
var i = 1
let argv = CommandLine.arguments
while i < argv.count {
    let a = argv[i]
    switch a {
    case "--expected-frames": i += 1; args.expectedFrames = Int(argv[i]) ?? 24
    case "--expected-duration": i += 1; args.expectedDuration = Double(argv[i]) ?? 3.0
    case "--mode": i += 1; args.mode = argv[i]
    case "--baseline": i += 1; args.baseline = argv[i]
    case "--alpha": i += 1; args.alpha = Double(argv[i]) ?? 0.3
    case "--emit-baseline": args.emitBaseline = true
    default: if args.path.isEmpty { args.path = a } else { die("unexpected arg: \(a)") }
    }
    i += 1
}
if args.path.isEmpty { die("usage: validate_gif.swift <path> [...]") }

let url = URL(fileURLWithPath: args.path)
guard let data = try? Data(contentsOf: url) else { die("read failed: \(args.path)") }

// -- Check 1: GIF89a header
let header = data.prefix(6)
guard header == Data("GIF89a".utf8) else { die("header mismatch: \(header.map { String(format: "%02x", $0) }.joined())") }
print("ok: header GIF89a")

// -- Check 2: Frame count
guard let src = CGImageSourceCreateWithData(data as CFData, nil) else { die("CGImageSourceCreateWithData failed") }
let n = CGImageSourceGetCount(src)
if n < args.expectedFrames { die("frames: got \(n), expected >= \(args.expectedFrames)") }
print("ok: \(n) frames")

// -- Check 3: Duration within ±5%
var total: Double = 0
for f in 0..<n {
    let props = CGImageSourceCopyPropertiesAtIndex(src, f, nil) as? [CFString: Any]
    let gifDict = props?[kCGImagePropertyGIFDictionary] as? [CFString: Any]
    let delay = (gifDict?[kCGImagePropertyGIFDelayTime] as? Double) ?? 0.1
    total += delay
}
let lower = args.expectedDuration * 0.95
let upper = args.expectedDuration * 1.05
if total < lower || total > upper {
    die("duration: got \(total), expected \(args.expectedDuration) ±5%")
}
print("ok: duration \(String(format: "%.3f", total))s")

// -- Check 4: Global color table bit (Best mode only)
// LSD packed field is at offset 10; bit 7 is the GCT flag.
if args.mode == "best" {
    let packed = data[10]
    if (packed & 0x80) == 0 { die("global color table bit not set (Best mode requires GCT)") }
    print("ok: global color table present")
}

// -- Check 5: Flicker metric
// Sample a 32x32 region at the center across all frames.
// Variance of palette-quantized RGB across frames, averaged per pixel.
let REGION = 32
var samples: [[SIMD3<Double>]] = [] // per frame: REGION*REGION values
for f in 0..<n {
    guard let cg = CGImageSourceCreateImageAtIndex(src, f, nil) else { die("decode frame \(f)") }
    let W = cg.width, H = cg.height
    let x0 = (W - REGION) / 2, y0 = (H - REGION) / 2
    let bpr = REGION * 4
    var buf = [UInt8](repeating: 0, count: bpr * REGION)
    let ctx = CGContext(
        data: &buf, width: REGION, height: REGION,
        bitsPerComponent: 8, bytesPerRow: bpr,
        space: CGColorSpaceCreateDeviceRGB(),
        bitmapInfo: CGImageAlphaInfo.premultipliedLast.rawValue
    )!
    ctx.translateBy(x: -CGFloat(x0), y: -CGFloat(y0))
    ctx.draw(cg, in: CGRect(x: 0, y: 0, width: W, height: H))
    var pixels = [SIMD3<Double>](); pixels.reserveCapacity(REGION * REGION)
    for p in stride(from: 0, to: buf.count, by: 4) {
        pixels.append(SIMD3<Double>(Double(buf[p]), Double(buf[p+1]), Double(buf[p+2])))
    }
    samples.append(pixels)
}
let K = REGION * REGION
var flicker: Double = 0
for p in 0..<K {
    var mean = SIMD3<Double>(0, 0, 0)
    for s in samples { mean += s[p] }
    mean /= Double(n)
    var variance: Double = 0
    for s in samples {
        let d = s[p] - mean
        variance += (d.x*d.x + d.y*d.y + d.z*d.z)
    }
    flicker += variance / Double(n)
}
flicker /= Double(K)
print("flicker: \(String(format: "%.4f", flicker))")

// Two modes:
// 1. --emit-baseline writes the flicker value to the baseline file + hash.
// 2. default reads the baseline and asserts flicker <= max(alpha * B0, 0.5).
if args.emitBaseline {
    let b = "B0=\(flicker)\n"
    try b.write(toFile: args.baseline, atomically: true, encoding: .utf8)
    // Sign with git hash-object for tamper check.
    let p = Process()
    p.launchPath = "/usr/bin/env"
    p.arguments = ["git", "hash-object", args.baseline]
    let pipe = Pipe()
    p.standardOutput = pipe
    try p.run(); p.waitUntilExit()
    let hash = String(data: pipe.fileHandleForReading.readDataToEndOfFile(), encoding: .utf8)?
        .trimmingCharacters(in: .whitespacesAndNewlines) ?? ""
    try (b + "HASH=\(hash)\n").write(toFile: args.baseline, atomically: true, encoding: .utf8)
    print("ok: emitted baseline B0=\(flicker) hash=\(hash)")
    exit(0)
}

guard let baselineText = try? String(contentsOfFile: args.baseline) else {
    die("missing baseline: \(args.baseline) — run with --emit-baseline first")
}
let b0Line = baselineText.split(separator: "\n").first { $0.hasPrefix("B0=") }
guard let b0Str = b0Line?.split(separator: "=").last, let B0 = Double(b0Str) else {
    die("baseline unreadable")
}
let bound = max(args.alpha * B0, 0.5)
if flicker > bound {
    die("flicker \(flicker) > bound \(bound) (alpha=\(args.alpha), B0=\(B0))")
}
print("ok: flicker \(String(format: "%.4f", flicker)) <= bound \(String(format: "%.4f", bound)) (alpha=\(args.alpha), B0=\(B0))")
```

```rust
/// Runs one-frame NeuQuant on `rgba` and returns a RawFrame whose pixels are
/// the palette-reconstructed image. Preserves alpha. Caller frees with
/// `fastgif_raw_frame_free`.
///
/// This is the witness for §9.P3 (preview_parity): it goes through the
/// same `NeuQuant::new` + `index_of` pipeline as `fastgif_encode` in draft
/// mode, so `preview_color_budget ≡ export_color_budget(.draft)` by construction.
///
/// # Safety
/// - `rgba` must point to `w * h * 4` bytes.
#[no_mangle]
pub unsafe extern "C" fn fastgif_preview_frame(
    rgba: *const u8,
    w: u32,
    h: u32,
    colors: u32,
) -> *mut RawFrame {
    if rgba.is_null() || w == 0 || h == 0 {
        return ptr::null_mut();
    }
    let n_px = (w as usize) * (h as usize);
    let input = slice::from_raw_parts(rgba, n_px * 4);
    let num_colors = (colors as usize).clamp(2, 256);
    let nq = color_quant::NeuQuant::new(10, num_colors, input);

    // Build palette-reconstructed RGBA.
    let mut out_pixels = vec![0u8; n_px * 4];
    let palette = nq.color_map_rgb();
    for (dst, src) in out_pixels.chunks_exact_mut(4).zip(input.chunks_exact(4)) {
        let idx = nq.index_of(src);
        let base = idx * 3;
        dst[0] = palette[base];
        dst[1] = palette[base + 1];
        dst[2] = palette[base + 2];
        dst[3] = src[3]; // preserve alpha
    }

    // Leak the vec as a raw pointer; free in fastgif_raw_frame_free.
    let mut boxed = out_pixels.into_boxed_slice();
    let rgba_ptr = boxed.as_mut_ptr();
    std::mem::forget(boxed);

    Box::into_raw(Box::new(RawFrame {
        rgba: rgba_ptr,
        width: w,
        height: h,
        delay_cs: 0,
    }))
}

/// Free a RawFrame returned by `fastgif_preview_frame`.
///
/// # Safety
/// - `rf` must be a pointer returned by `fastgif_preview_frame`, or null.
#[no_mangle]
pub unsafe extern "C" fn fastgif_raw_frame_free(rf: *mut RawFrame) {
    if rf.is_null() { return; }
    let boxed = Box::from_raw(rf);
    let n = (boxed.width as usize) * (boxed.height as usize) * 4;
    drop(Vec::from_raw_parts(boxed.rgba as *mut u8, n, n));
}
```

```swift
/// 8x8 ordered Bayer dither — real, deterministic, GPU. Adds ±1-quant-step
/// noise before NeuQuant sees the pixels, which breaks up banding in flat
/// regions without changing the palette.
struct BayerDither: Stage {
    static let kernel: CIColorKernel = {
        let src = """
        float bayer8(float2 p) {
            int x = int(p.x) & 7;
            int y = int(p.y) & 7;
            int i = y * 8 + x;
            float m[64] = {
                 0,32, 8,40, 2,34,10,42,
                48,16,56,24,50,18,58,26,
                12,44, 4,36,14,46, 6,38,
                60,28,52,20,62,30,54,22,
                 3,35,11,43, 1,33, 9,41,
                51,19,59,27,49,17,57,25,
                15,47, 7,39,13,45, 5,37,
                63,31,55,23,61,29,53,21
            };
            return m[i] / 64.0 - 0.5;
        }
        kernel vec4 bayerDither(__sample s, destination dest) {
            float offset = bayer8(dest.coord()) / 256.0;
            return vec4(s.rgb + vec3(offset), s.a);
        }
        """
        guard let k = CIColorKernel(source: src) else {
            fatalError("BayerDither: kernel compile failed")
        }
        return k
    }()

    func process(_ frames: [Frame]) async throws -> [Frame] {
        let ctx = CIContext(options: [.useSoftwareRenderer: false])
        var out = [Frame]()
        out.reserveCapacity(frames.count)
        for frame in frames {
            let processed: Frame = autoreleasepool {
                let ci = CIImage(cgImage: frame.image)
                let args: [Any] = [ci]
                guard let dithered = BayerDither.kernel.apply(
                    extent: ci.extent, arguments: args
                ) else { return frame }
                guard let cg = ctx.createCGImage(dithered, from: dithered.extent) else { return frame }
                return Frame(image: cg, delay: frame.delay)
            }
            out.append(processed)
            await Task.yield()
        }
        return out
    }
}
```

```swift
    /// Run `body` with an array of RawFrame structs whose pixel buffers live
    /// for the duration of the closure. Handles all ownership.
    private static func withRawFrames<R>(
        _ frames: [Frame],
        _ body: (UnsafeBufferPointer<RawFrame>) throws -> R
    ) throws -> R {
        guard !frames.isEmpty else { throw EncoderError.noFrames }
        var buffers: [UnsafeMutablePointer<UInt8>] = []
        defer { buffers.forEach { $0.deallocate() } }
        var rawFrames: [RawFrame] = []
        rawFrames.reserveCapacity(frames.count)
        for frame in frames {
            let w = frame.image.width, h = frame.image.height
            let buf = UnsafeMutablePointer<UInt8>.allocate(capacity: w * h * 4)
            guard let ctx = CGContext(
                data: buf,
                width: w, height: h,
                bitsPerComponent: 8,
                bytesPerRow: w * 4,
                space: CGColorSpaceCreateDeviceRGB(),
                bitmapInfo: CGImageAlphaInfo.premultipliedLast.rawValue
            ) else { buf.deallocate(); continue }
            ctx.draw(frame.image, in: CGRect(x: 0, y: 0, width: w, height: h))
            rawFrames.append(RawFrame(
                rgba: UnsafePointer(buf),
                width: UInt32(w),
                height: UInt32(h),
                delay_cs: UInt16(max(2, Int(frame.delay * 100)))
            ))
            buffers.append(buf)
        }
        guard !rawFrames.isEmpty else { throw EncoderError.noFrames }
        return try rawFrames.withUnsafeBufferPointer(body)
    }

        frames: [Frame],
        loopCount: Int = 0,

                UInt16(loopCount),
                quality == .draft ? 30 : 10
            ) else { throw EncoderError.finalizeFailed }
            defer { fastgif_free(result) }
            return Data(bytes: result.pointee.data, count: result.pointee.len)

    /// This is the construction witness for §9.P3 (preview_parity):
    /// the preview color set is produced by the same NeuQuant::new +
    /// index_of pipeline that encodeGIF(quality: .draft) uses.
    static func previewQuantize(frame: Frame, colors: Int) -> Frame {
        let w = frame.image.width, h = frame.image.height
        let nBytes = w * h * 4

        // Draw the source CGImage into a contiguous RGBA buffer for Rust.
```

```rust
fn palette_error(palette_rgb: &[u8], clip: &[Vec<u8>]) -> f64 {
    let colors = palette_rgb.len() / 3;
    let mut total = 0f64;
    let mut n = 0u64;
    for frame in clip {
        for px in frame.chunks_exact(4) {
            let (r, g, b) = (px[0] as i32, px[1] as i32, px[2] as i32);
            let mut best = i32::MAX;
            for k in 0..colors {
                let pr = palette_rgb[k * 3] as i32;
                let pg = palette_rgb[k * 3 + 1] as i32;
                let pb = palette_rgb[k * 3 + 2] as i32;
                let d = (r - pr).pow(2) + (g - pg).pow(2) + (b - pb).pow(2);
                if d < best { best = d; }
            }
            total += best as f64;
            n += 1;
        }
    }
    total / n as f64
}

    let mut concat = Vec::with_capacity(sample_indices.len() * 240 * 240 * 4);
    for &i in sample_indices {
        concat.extend_from_slice(&frames[i]);
    }
    color_quant::NeuQuant::new(10, 256, &concat).color_map_rgb()
}
```

```rust
//! Sierra2_4a spatial error diffusion — the zero-flicker kernel.
//!
//! The error buffer is passed in by caller so the temporal-carry variant
//! (commit 4) can thread E_{n-1} → E_n. For commit 3 it is always
//! zero-seeded — the "spatial only" flavor.
//!
//! Weights (Sierra2_4a):
//!
//!        *  2/4
//!  1/4  1/4
//!
//! Distributes the quantization error from pixel (x,y) to its three
//! unvisited neighbors. Cheaper than Floyd-Steinberg, visually comparable.

use color_quant::NeuQuant;

/// Run sequential (raster-order) Sierra2_4a diffusion over `rgba`.
/// `err_in_out` is `w*h` slots of [i16; 3]; treated as in/out.
/// Returns palette indices (length `w*h`).
pub fn diffuse_sequential(
    rgba: &[u8],
    w: usize,
    h: usize,
    nq: &NeuQuant,
    err_in_out: &mut [[i16; 3]],
) -> Vec<u8> {
    assert_eq!(rgba.len(), w * h * 4);
    assert_eq!(err_in_out.len(), w * h);
    let mut out = vec![0u8; w * h];
    for y in 0..h {
        for x in 0..w {
            let i = y * w + x;
            let e = err_in_out[i];
            // Apply incoming error to this pixel.
            let r = (rgba[i*4+0] as i16 + e[0]).clamp(0, 255) as u8;
            let g = (rgba[i*4+1] as i16 + e[1]).clamp(0, 255) as u8;
            let b = (rgba[i*4+2] as i16 + e[2]).clamp(0, 255) as u8;
            let a = rgba[i*4+3];
            let rgba_pixel = [r, g, b, a];
            let idx = nq.index_of(&rgba_pixel);
            out[i] = idx as u8;

            // Compute quantization error = actual - palette[idx].
            let pal = nq.color_map_rgb();
            let pr = pal[idx*3] as i16;
            let pg = pal[idx*3+1] as i16;
            let pb = pal[idx*3+2] as i16;
            let dr = r as i16 - pr;
            let dg = g as i16 - pg;
            let db = b as i16 - pb;

            // Sierra2_4a weights (denominator 4): right 2, down-left 1, down 1.
            let distribute = |dst: &mut [i16; 3], k: i16| {
                dst[0] = dst[0].saturating_add(dr * k / 4);
                dst[1] = dst[1].saturating_add(dg * k / 4);
                dst[2] = dst[2].saturating_add(db * k / 4);
            };
            if x + 1 < w { distribute(&mut err_in_out[y * w + (x + 1)], 2); }
            if y + 1 < h {
                if x > 0     { distribute(&mut err_in_out[(y+1) * w + (x-1)], 1); }
                              distribute(&mut err_in_out[(y+1) * w + x],      1);
            }
        }
    }
    out
}
```

```rust
/// Global-palette GIF encoder with spatial Sierra2_4a (commit 3) and
/// temporal carry (commit 4). See §1.3–§1.4.
///
/// # Safety
/// Same contract as `fastgif_encode`.
#[no_mangle]
pub unsafe extern "C" fn fastgif_encode_global(
    frames_ptr: *const RawFrame,
    count: usize,
    colors: u32,
    loop_count: u16,
    quality: i32,
) -> *mut GIFOutput {
    if frames_ptr.is_null() || count == 0 {
        return ptr::null_mut();
    }
    let raw_frames = slice::from_raw_parts(frames_ptr, count);
    let w = raw_frames[0].width;
    let h = raw_frames[0].height;
    let num_colors = (colors as usize).clamp(2, 256);
    let sample_fac = quality.clamp(1, 30);

    // 1. Sample 8 evenly spaced frames (or all, if count < 8).
    let sample_count = 8usize.min(count);
    let sample_indices: Vec<usize> = (0..sample_count)
        .map(|i| (i * count) / sample_count)
        .collect();
    let mut training = Vec::with_capacity(sample_count * (w * h * 4) as usize);
    for &i in &sample_indices {
        let rf = &raw_frames[i];
        let pixels = slice::from_raw_parts(rf.rgba, (rf.width * rf.height * 4) as usize);
        training.extend_from_slice(pixels);
    }

    // 2. Train one NeuQuant on the concatenated training buffer.
    let nq = color_quant::NeuQuant::new(sample_fac, num_colors, &training);
    let global_palette = nq.color_map_rgb();

    // 3. Build encoder with global palette.
    let mut buf: Vec<u8> = Vec::with_capacity(count * (w * h) as usize);
    let result = (|| -> Result<(), gif::EncodingError> {
        let mut enc = gif::Encoder::new(&mut buf, w as u16, h as u16, &global_palette)?;
        enc.set_repeat(if loop_count == 0 { gif::Repeat::Infinite } else { gif::Repeat::Finite(loop_count) })?;

        // 4. Diffuse each frame. Commit 3 zero-seeds `err_carry` per frame
        //    (spatial only); commit 4 threads it across frames.
        for rf in raw_frames {
            let n_px = (rf.width as usize) * (rf.height as usize);
            let rgba = slice::from_raw_parts(rf.rgba, n_px * 4);
            let mut err = vec![[0i16; 3]; n_px];
            let indices = diffuse::diffuse_sequential(
                rgba, rf.width as usize, rf.height as usize, &nq, &mut err
            );
            let mut frame = gif::Frame::default();
            frame.width = rf.width as u16;
            frame.height = rf.height as u16;
            frame.delay = rf.delay_cs;
            frame.palette = None; // use global
            frame.buffer = std::borrow::Cow::Owned(indices);
            enc.write_frame(&frame)?;
        }
        Ok(())
    })();

    if result.is_err() { return ptr::null_mut(); }
    let len = buf.len();
    let data = {
        let mut boxed = buf.into_boxed_slice();
        let p = boxed.as_mut_ptr();
        std::mem::forget(boxed);
        p
    };
    Box::into_raw(Box::new(GIFOutput { data, len }))
```

```rust
pub fn diffuse_tiled(
    rgba: &[u8],
    w: usize,
    h: usize,
    nq: &NeuQuant,
    err_in_out: &mut [[i16; 3]],
    num_tiles: usize,
) -> Vec<u8> {
    assert!(num_tiles >= 1);
    if num_tiles == 1 || h < num_tiles * 4 {
        return diffuse_sequential(rgba, w, h, nq, err_in_out);
    }

    // Contiguous tile heights; last tile absorbs remainder.
    let base = h / num_tiles;
    let mut starts = Vec::with_capacity(num_tiles + 1);
    for t in 0..num_tiles { starts.push(t * base); }
    starts.push(h);

    let mut out = vec![0u8; w * h];
    let progress: Vec<AtomicUsize> =
        (0..num_tiles).map(|_| AtomicUsize::new(0)).collect();

    // Safety: each thread writes to disjoint row ranges in `out` and
    // `err_in_out`. The barrier guarantees tile t only reads rows above
    // its own start AFTER tile t-1 has finalized them.
    crossbeam_utils::thread::scope(|s| {
        // Split err_in_out and out into disjoint per-tile mutable slices.
        // This requires `unsafe` because the rows a tile writes during
        // diffusion also appear as the "from above" footprint for the tile
        // below — but the barrier proves that interleave is safe.
        let out_ptr = out.as_mut_ptr() as usize;
        let err_ptr = err_in_out.as_mut_ptr() as usize;

        for t in 0..num_tiles {
            let progress = &progress;
            let starts = &starts;
            s.spawn(move |_| {
                let y0 = starts[t];
                let y1 = starts[t + 1];
                for y in y0..y1 {
                    if t > 0 {
                        // Sierra2_4a row `y` depends on all of row `y-1` being
                        // committed (because `(x+1, y-1)` writes to `(x, y)`
                        // via the down-left diffusion term). Tile t-1 stores
                        // its progress as "rows committed from its start."
                        // Condition: tile t-1 must have committed enough rows
                        // that its last committed row is >= y - 1, i.e.
                        //   progress[t-1] >= y - starts[t-1].
                        // This is a 1-row lag between tiles. sierra_parity.rs
                        // will fail loudly if the boundary is wrong — adjust
                        // the condition until the test passes on T ∈ {1..8}.
                        let needed = y - starts[t - 1];
                        while progress[t-1].load(Ordering::Acquire) < needed {
                            std::hint::spin_loop();
                        }
                    }
                    // Run a single-row diffusion. Mirrors diffuse_sequential's
                    // inner body restricted to row `y`.
                    unsafe {
                        let out_slice = std::slice::from_raw_parts_mut(
                            out_ptr as *mut u8, w * h);
                        let err_slice = std::slice::from_raw_parts_mut(
                            err_ptr as *mut [i16; 3], w * h);
                        diffuse_one_row(rgba, w, h, y, nq, err_slice, out_slice);
                    }
                    progress[t].store(y - y0 + 1, Ordering::Release);
                }
            });
        }
    }).expect("diffuse_tiled threads panicked");
    out
}

fn diffuse_one_row(
    rgba: &[u8],
    w: usize,
    h: usize,
    y: usize,
    nq: &NeuQuant,
    err: &mut [[i16; 3]],
    out: &mut [u8],
) {
    for x in 0..w {
        let i = y * w + x;
        let e = err[i];
        let r = (rgba[i*4+0] as i16 + e[0]).clamp(0, 255) as u8;
        let g = (rgba[i*4+1] as i16 + e[1]).clamp(0, 255) as u8;
        let b = (rgba[i*4+2] as i16 + e[2]).clamp(0, 255) as u8;
        let a = rgba[i*4+3];
        let pixel = [r, g, b, a];
        let idx = nq.index_of(&pixel);
        out[i] = idx as u8;
        let pal = nq.color_map_rgb();
        let dr = r as i16 - pal[idx*3]   as i16;
        let dg = g as i16 - pal[idx*3+1] as i16;
        let db = b as i16 - pal[idx*3+2] as i16;
        let distribute = |dst: &mut [i16; 3], k: i16| {
            dst[0] = dst[0].saturating_add(dr * k / 4);
            dst[1] = dst[1].saturating_add(dg * k / 4);
            dst[2] = dst[2].saturating_add(db * k / 4);
        };
        if x + 1 < w { distribute(&mut err[y * w + (x + 1)], 2); }
        if y + 1 < h {
            if x > 0     { distribute(&mut err[(y+1) * w + (x-1)], 1); }
                          distribute(&mut err[(y+1) * w + x],      1);
        }
    }
}
```

Barrier derivation (kept as an in-source comment so nobody has to re-derive): Sierra2_4a from pixel `(x, y-1)` can write to `(x, y)` (via the `(px, py+1)` term) AND `(x-1, y)` (via the `(px-1, py+1)` term). The `(x+1, y-1)` write to `(x, y)` via `(px-1, py+1)` means that *every* column of row `y-1` may still touch row `y` until it's processed. Therefore row `y` of tile `t` requires row `y-1` of tile `t-1` to be fully committed — a one-row lag. The `progress[t-1]` counter is "rows committed from tile `t-1`'s start," so the condition is `progress[t-1] >= y - starts[t-1]`. If `sierra_parity.rs` fails at `T > 1`, the first thing to check is this boundary — try `>= y - starts[t-1] + 1` as a conservative widening before touching anything else.

```swift
import SwiftUI

/// Unified timeline: one horizontal row for trim + scrub + time label.
/// Replaces TrimView + TimeScrubber. Owns one DragGesture whose hit-testing
/// at drag-begin decides which handle (start, end, playhead) captures it.
struct Timeline: View {
    @Bindable var project: GIFProject
    let onRetrim: () -> Void

    private enum DragTarget { case start, end, playhead }
    @State private var drag: DragTarget?

    private var trimStartFrac: Double {
        project.videoDuration > 0 ? project.trimStart / project.videoDuration : 0
    }
    private var trimEndFrac: Double {
        project.videoDuration > 0 ? (project.trimEnd ?? project.videoDuration) / project.videoDuration : 1
    }
    private var playheadFrac: Double {
        let dur = (project.trimEnd ?? project.videoDuration) - project.trimStart
        return dur > 0 ? (project.currentTime / dur).clamped(to: 0...1) : 0
    }

    var body: some View {
        VStack(spacing: Theme.spacing4) {
            HStack {
                Text(formatTime(project.currentTime))
                    .font(.system(size: 13, design: .monospaced).weight(.medium))
                    .foregroundStyle(Theme.textPrimary)
                Spacer()
                Text("\(project.frames.count) frames")
                    .font(.system(size: 11, design: .monospaced))
                    .foregroundStyle(Theme.textTertiary)
                Spacer()
                Text(formatTime((project.trimEnd ?? project.videoDuration) - project.trimStart))
                    .font(.system(size: 13, design: .monospaced).weight(.medium))
                    .foregroundStyle(Theme.textSecondary)
            }
            .padding(.horizontal, Theme.spacing16)

            GeometryReader { geo in
                let w = geo.size.width
                ZStack(alignment: .leading) {
                    Capsule()
                        .fill(Theme.surface)
                        .frame(height: 36)

                    // Trim region
                    Capsule()
                        .fill(Theme.accent.opacity(0.3))
                        .frame(width: CGFloat(trimEndFrac - trimStartFrac) * w, height: 36)
                        .offset(x: CGFloat(trimStartFrac) * w)

                    // Start handle
                    handle()
                        .offset(x: CGFloat(trimStartFrac) * w - 8)
                        .accessibilityLabel("Trim start")
                        .accessibilityValue(formatTime(project.trimStart))

                    // End handle
                    handle()
                        .offset(x: CGFloat(trimEndFrac) * w - 8)
                        .accessibilityLabel("Trim end")
                        .accessibilityValue(formatTime(project.trimEnd ?? project.videoDuration))

                    // Playhead
                    Circle()
                        .fill(Theme.accent)
                        .frame(width: 14, height: 14)
                        .shadow(color: Theme.accent.opacity(0.3), radius: 4)
                        .offset(x: trimStartFrac.interp(to: trimEndFrac, frac: playheadFrac) * w - 7)
                }
                .contentShape(Rectangle())
                .gesture(
                    DragGesture(minimumDistance: 0)
                        .onChanged { v in
                            if drag == nil { drag = pickTarget(at: v.startLocation.x, width: w) }
                            applyDrag(value: v, width: w)
                        }
                        .onEnded { v in
                            applyDrag(value: v, width: w)
                            if drag != .playhead { onRetrim() }
                            drag = nil
                        }
                )
            }
            .frame(height: 36)
            .padding(.horizontal, Theme.spacing16)
        }
        .padding(.vertical, Theme.spacing8)
        .background(Theme.surface)
    }

    private func handle() -> some View {
        RoundedRectangle(cornerRadius: 3)
            .fill(Theme.accent)
            .frame(width: 16, height: 44)
    }

    private func pickTarget(at x: CGFloat, width: CGFloat) -> DragTarget {
        let startX = CGFloat(trimStartFrac) * width
        let endX = CGFloat(trimEndFrac) * width
        let playheadX = trimStartFrac.interp(to: trimEndFrac, frac: playheadFrac) * width
        let candidates: [(DragTarget, CGFloat)] = [
            (.start, abs(x - startX)),
            (.end, abs(x - endX)),
            (.playhead, abs(x - playheadX) + 4), // handles win ties
        ]
        let within = candidates.filter { $0.1 < 20 }
        return within.min(by: { $0.1 < $1.1 })?.0 ?? .playhead
    }

    private func applyDrag(value: DragGesture.Value, width: CGFloat) {
        let frac = Double(value.location.x / width).clamped(to: 0...1)
        let t = frac * project.videoDuration
        switch drag {
        case .start:
            project.trimStart = min(t, (project.trimEnd ?? project.videoDuration) - 0.05)
        case .end:
            project.trimEnd = max(t, project.trimStart + 0.05)
        case .playhead, nil:
            let range = (project.trimEnd ?? project.videoDuration) - project.trimStart
            let playheadRelative = (frac - trimStartFrac)
                .clamped(to: 0...(trimEndFrac - trimStartFrac))
            let normalized = (trimEndFrac - trimStartFrac) > 0
                ? playheadRelative / (trimEndFrac - trimStartFrac)
                : 0
            project.currentTime = normalized * range
        }
    }

    private func formatTime(_ s: Double) -> String {
        let whole = Int(s)
        let ms = Int((s.truncatingRemainder(dividingBy: 1)) * 10)
        return whole >= 60
            ? String(format: "%d:%02d.%d", whole / 60, whole % 60, ms)
            : String(format: "%d.%ds", whole, ms)
    }
}

private extension Double {
    func clamped(to range: ClosedRange<Double>) -> Double {
        Swift.min(Swift.max(self, range.lowerBound), range.upperBound)
    }
    func interp(to other: Double, frac: Double) -> CGFloat {
        CGFloat(self + (other - self) * frac)
    }
}
```
